History of the Web Browser

The first browser and web page editor was WorldWideWeb, created by Tim Berners-Lee in late 1990. It only worked on NeXT computers at CERN. The Line Mode Browser followed and was important because it could run on different operating systems. :contentReference[oaicite:6]{index=6}

Browser Wars

In 1994, Netscape Navigator became popular because it could display text and graphics while the page was still loading. In 1995, Microsoft released Internet Explorer with Windows, beginning the browser wars between Netscape and Microsoft. :contentReference[oaicite:7]{index=7}

Modern Browsers

  • Safari was first released in 2003
  • Opera has existed since 1994
  • Google Chrome was first released in 2008
  • Microsoft later moved from Internet Explorer to Edge
