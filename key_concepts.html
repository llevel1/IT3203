Key Concepts

1. World Wide Web

The web is a collection of documents based on hypertext that can be transported between computers through the internet.

2. HTML

HTML stands for HyperText Markup Language. It is one of the main elements of the web and provides the structure of web pages.

3. Browser

A browser is an application used to retrieve and present content from websites to users.

4. URL

URL stands for Uniform Resource Locator. It is the web address the user enters into the browser.

5. HTTP

HTTP stands for Hypertext Transfer Protocol. It is the protocol browsers use to send and receive information from servers.
