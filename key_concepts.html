Important Terms

This page explains 6 key concepts related to the web.

Key Concepts

1. Internet

The internet is the global network of connected computers and devices.

2. World Wide Web

The web is a system of linked pages and resources accessed over the internet.

3. HTML

HTML stands for HyperText Markup Language and provides the structure of web pages.

4. CSS

CSS stands for Cascading Style Sheets and controls the appearance and layout of web pages.

5. URL

A URL is the address used to locate a resource on the web.

6. HTTP

HTTP is the protocol used to transfer web pages and related resources between servers and browsers.

These concepts form the foundation for understanding how websites are built and accessed.
Concept Meaning
Internet Global network connecting devices
Web Collection of linked information pages
HTML Page structure language
CSS Page styling language
