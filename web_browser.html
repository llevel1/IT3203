Browser Facts

  • Browsers retrieve web pages
  • They display HTML to users
  • They use HTTP requests
  • They can use cached data

What Is a Web Browser?

A web browser is the application that lets users access information on websites. When a user types a URL into the address bar, the browser gets the files from a web server or from the cache and displays them on the device screen.

Main Browser Components

  • User interface
  • Browser engine
  • Rendering engine
  • Networking
  • UI backend
  • JavaScript interpreter
  • Data storage

How a Browser Works

  1. The user enters a URL or search term
  2. The browser checks whether it is a search or a URL
  3. The cache is checked first
  4. If needed, the browser performs a DNS lookup
  5. The browser connects to the server and sends a request
  6. The response is received and prepared for rendering
  7. The page is displayed to the user
Component Purpose
User Interface The visible parts like buttons and address bar
Rendering Engine Displays the content on the screen
Networking Sends and receives HTTP requests
Data Storage Stores local data like bookmarks and cookies
