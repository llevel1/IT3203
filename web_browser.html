Browser Basics

  • Requests web pages
  • Interprets HTML and CSS
  • Displays content for users

What Is a Web Browser?

A web browser is a software application used to access websites and online resources. It sends requests to web servers, receives data, and displays the results in a readable format.

Examples of Browsers

  • Google Chrome
  • Mozilla Firefox
  • Microsoft Edge
  • Safari

What Browsers Do

  1. Read the URL entered by the user
  2. Send a request to the appropriate server
  3. Download HTML, CSS, images, and scripts
  4. Render the page for the user
Browser Main Use
